# Architecture — Unreal Reels: A Practitioner's Guide

## Learning model

Problem-first, whole-task (Merrill). Every chapter is a short overview (~1–3 pages on *why* the pipeline works this way) followed by exercises that each produce an artifact. No comprehension-only chapters — every chapter targets Apply or above on Bloom's.

## Three-act arc

- **Act I — Foundations (Ch 1–3):** the two laws, the beat sheet, the storyboard engine. The reader learns the invariants before any genre.
- **Act II — Making videos (Ch 4–7):** the three reels aspects plus the lecture pipeline. The reader makes one of each.
- **Act III — Scale & trust (Ch 8–12):** silent mode, fidelity, the look, the ecosystem, and refinement. The reader goes from one video to a whole book's worth, and learns what only they can decide.

## Outcome map (Bloom's)

| Ch | Capability built | Level |
|---|---|---|
| 1 | Explain why audio-first + gates control quality; identify a broken gate | Understand/Analyze |

| 3 | Run the storyboard engine to a first still set; pick keepers | Apply |
| 4 | Produce a lyric/music video (or kids' language video) | Apply |
| 5 | Produce a mini-biography | Apply |

| 8 | Run silent mode over several chapters; read the triage manifest | Apply/Evaluate |
| 9 | Run the grounding check; judge a fidelity flag | Evaluate |
| 10 | Apply the design system; fix a static-screen violation | Apply |
| 11 | Trace one claim from book → fact commons → video; sign a gate | Evaluate |
| 12 | Refine a lecture into your own voice; ship it | Create/Evaluate |

## Assessment plan

Each chapter's exercises are the assessment; the artifact is the evidence. Optional editions (quizzes, flashcards, Canvas export) are produced with the AI+1 build tools after the spine is done. The book itself is a candidate for the lecture pipeline — a pleasing self-host, like `ai1-cli`.

## Sequencing rule

Foundations before genres; genres before scale. A reader who tries silent mode (Ch 8) without the beat sheet (Ch 2) will not understand why swapping audio re-times everything for free.
